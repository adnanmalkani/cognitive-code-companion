# Changelog

All notable changes to "Cognitive Code Companion" will be documented in this file.

## [0.1.0] - 2026-04-06

### Added

- "Explain This Code" command with right-click context menu integration
- Four cognitive explanation modes: Step-by-Step, Big Picture First, Analogy-Based, Minimal
- Sidebar webview panel with mode switching dropdown
- Claude API integration (claude-sonnet-4-20250514)
- Multi-language explanation support
- VS Code theme-aware UI (light/dark)
- Graceful error handling for missing API key and API failures
